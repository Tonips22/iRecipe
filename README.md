## Technologies

**Client:** HTML, CSS, SASS, JavaScript.

**Server:** PHP, MySQL.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Tonips22/iRecipe.git
```

Go to the project directory

```bash
  cd iRecipe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
